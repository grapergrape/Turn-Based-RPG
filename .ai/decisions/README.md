# Decisions

This folder stores ADR-style records for important project decisions.

## When to Add a Decision

Add a decision record when choosing or changing:

- Engine or framework
- Repository structure
- Core gameplay systems
- Narrative canon or world structure
- Asset pipeline or naming conventions
- Testing or build process

## Naming

Use `NNNN-short-kebab-case-title.md`, starting at `0001`.
