# `assets/` Asset Guide

Static game assets live here.

Suggested folders:

```text
assets/
├── sprites/
├── tilesets/
├── audio/
└── ui/
```

## Rules

- Use descriptive kebab-case names.
- Do not use spaces in file names.
- Do not put source code here.
- Do not put game content JSON here unless it is asset metadata.
- Keep pixel-art scale intentional.
- Export game-ready assets separately from raw working files.

## Examples

```text
assets/sprites/player-militia-idle.png
assets/sprites/host-penitent-bastion-idle.png
assets/tilesets/ruined-city-01.png
assets/audio/ui-select.ogg
assets/ui/dialogue-frame.png
```
