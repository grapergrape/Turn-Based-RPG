# Game Documentation

This directory contains design and production documentation for the turn-based RPG.

## Sections

- `story/` — worldbuilding, characters, quests, factions, and narrative direction.
- `lore/` — canon bible and high-authority setting references.
- `enemies/` — enemy families, encounter roles, bosses, and combat behaviors.
- `items/` — weapons, armor, consumables, key items, loot, and economy notes.
- `systems/` — combat, progression, inventory, dialogue, exploration, and other mechanics.
- `maps/` — regions, towns, dungeons, encounter zones, and traversal notes.
- `art-audio/` — visual style, animation, UI, music, and sound direction.
- `technical/` — engine, architecture, build, testing, and implementation notes.
- `templates/` — reusable documentation templates.

## Update Rule

When game content or systems change, update the relevant documentation in this directory and summarize durable facts in `.ai/memory/`.
