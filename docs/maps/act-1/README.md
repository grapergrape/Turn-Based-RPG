# Act I Map Packages

This index covers the Act I locations that currently have map-planning
artifacts or a locked child-map plan. Runtime levels without a planning package
remain discoverable under `data/levels/` and can be added here when they gain a
canonical spatial reference or submap registry.

| Route order | Main location | Runtime level | Package state |
| ---: | --- | --- | --- |
| 1 | [Long Ash Road Approach](./long-ash-road-approach/README.md) | `data/levels/long_ash_road_approach.json` | Main map plus nine implemented submaps |
| 2 | [Censure Road Camp](./censure-road-camp/README.md) | `data/levels/censure_road_camp.json` | Main map plus eight implemented tent interiors |
| 3 | [Ash Road South](./ash-road-south/README.md) | `data/levels/ash_road_south.json` | Main map plus nine implemented submaps |
| 4 | [Old Pilgrim Way](./old-pilgrim-way/README.md) | `data/levels/old_pilgrim_way.json` | Main map plus five implemented submaps |

The route order describes campaign geography, not automatic access. Individual
level transitions and dialogue conditions remain authoritative for when the
player can travel onward.

See the [map library rules](../README.md) before adding another package.
