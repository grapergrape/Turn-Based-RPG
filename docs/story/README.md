# Story Documentation

Use this folder for the game's narrative foundation.

## Suggested Files

- `world.md` — setting, regions, history, cultures, factions, and tone.
- `characters.md` — protagonists, party members, NPCs, villains, and relationships.
- `quests.md` — main quests, side quests, objectives, rewards, and branching outcomes.

Keep canonical story decisions synchronized with `.ai/memory/story.md`.
