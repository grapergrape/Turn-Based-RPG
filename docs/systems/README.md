# Systems Documentation

Use this folder to design and maintain gameplay systems.

## Files

- `combat.md` - turn structure, actions, targeting, damage, status effects, combat lanes, and encounter flow.
- `progression.md` - character sheets, primary attributes, field ratings, scars, Trace, Icon Risk, techniques, clearances, and rewards.
- `inventory.md` - item storage, equipment, stack rules, and economy interactions.
- `dialogue.md` - conversations, choices, branching, reputation, and consequences.

Summarize stable system decisions in `.ai/memory/systems.md` if that file is introduced later.
