# Systems Documentation

Use this folder to design and maintain gameplay systems.

## Files

- `combat.md` — turn structure, actions, targeting, damage, status effects, and encounter flow.
- `progression.md` — character sheets, Five Keeps, Seven Virtues, Trace, origins, field roles, scars, vows, liabilities, abilities, unlocks, and rewards.
- `inventory.md` — item storage, equipment, stack rules, and economy interactions.
- `dialogue.md` — conversations, choices, branching, reputation, and consequences.

Summarize stable system decisions in `.ai/memory/systems.md`.
