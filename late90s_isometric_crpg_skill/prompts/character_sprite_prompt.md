# Character Sprite Prompt

```text
Design an original late-1990s isometric CRPG character sprite set for [CHARACTER ROLE]. The character must be a small realistic pre-rendered-style human sprite, not chibi, not voxel, not modern HD pixel art.

Native target:
- standing height: 50–64 px
- width: 32–48 px including gear
- head: 7–10 px tall
- pivot: bottom-center between feet
- shadow: 24–36 px wide, 10–18 px tall
- facings: n, ne, e, se, s, sw, w, nw

Silhouette:
- [describe role silhouette]
- readable at gameplay zoom
- small head, narrow shoulders, layered clothing, clear boots, distinct arms
- asymmetrical gear: [bag/holster/strap/tool/weapon]

Palette:
- muted browns, ash gray, faded olive, leather, rust, dull metal
- one small accent only: [accent]
- limited color ramps, hard highlights, dithered shadows

Animations:
- idle: 4–8 frames, subtle breathing/weight shift
- walk: 8–12 frames per direction, 10–12 fps, planted feet, no sliding
- interact: 6–10 frames
- hurt: 3–6 frames
- death/corpse if killable: 10–18 frames plus corpse sprite

Rendering:
- fixed isometric orthographic view
- baked light from upper-left/front-left
- hard pixel clusters, no smooth gradients
- transparent background
- contact shadow included or separate

Do not copy any existing game character, armor, faction, mascot, or iconic outfit.
```
