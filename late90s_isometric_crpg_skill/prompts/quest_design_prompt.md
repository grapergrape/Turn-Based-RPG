# Quest Design Prompt

```text
Design an original late-1990s post-apocalyptic isometric CRPG quest for [LOCATION/FACTION]. The quest should support branching solutions, skill checks, environmental clues, moral ambiguity, and consequences.

Quest requirements:
- premise
- surface conflict
- hidden truth
- at least three solution paths
- at least one non-combat solution
- at least one combat path
- at least one stealth/technical/social path
- skill checks with meaningful consequences
- failure states
- short journal entries
- inspection text for key objects
- NPC dialogue options
- immediate and delayed consequences

Tone:
Dry, grim, practical, strange, slightly darkly funny. Avoid heroic prophecy, generic raiders, and clean good/evil morality.

Output format:
1. Quest title
2. Location summary
3. Key NPCs
4. Factions involved
5. Environmental clues
6. Solution paths
7. Skill checks
8. Dialogue sample
9. Journal entries
10. Consequences
11. Art/animation requirements

Do not copy quests, characters, factions, or dialogue from existing games.
```
