# UI Prompt

```text
Design an original late-1990s isometric CRPG interface. It should feel heavy, utilitarian, old, tactile, and slightly oppressive. Do not copy any existing game UI layout, device, icons, or mascot.

Target:
- low-resolution UI for 640x480 or 800x600 internal resolution
- dark metal/leather/brass panels
- aged-gold or rust borders
- compact bitmap font
- dull amber/tan/green-gray text
- small inset boxes
- subtle scratches, dust, rivets, worn corners
- no modern vector flat design
- no glossy mobile buttons

Required components:
- bottom message log
- character status panel
- HP/status readout
- action/mode display
- inventory access
- skill/action hotkeys
- dialogue panel style
- inventory grid or item list
- old cursor icons for move/inspect/talk/use/attack/loot

Readability:
- text readable at native resolution
- high enough contrast without bright white
- icon silhouettes clear at 16–32 px
- panels support dense information but do not hide gameplay unnecessarily

Negative style:
No Pip-Boy copy, no Vault-Tec copy, no exact Fallout buttons, no clean web UI, no neon sci-fi HUD, no transparent glassmorphism, no oversized mobile elements.
```
