# IP Safety Reference

## Goal

Create an original game that captures a late-1990s isometric CRPG production style without copying protected assets, designs, names, UI, characters, factions, maps, dialogue, or symbols from any existing game.

## Safe inspiration

Safe to study and emulate at a broad level:

- fixed isometric camera
- low internal resolution
- pre-rendered-style sprite workflow
- muted post-apocalyptic palette
- turn-based or tactical RPG structure
- branching dialogue
- skill checks
- grid or tile-aware movement
- heavy utilitarian UI
- old-school inventory density
- environmental storytelling
- dry dark humor
- consequence-driven quests

## Do not copy

Do not copy:

- exact UI panel layout from any existing game
- exact item icons
- exact cursor icons
- Pip-Boy-like device designs
- Vault Boy-style mascots
- Vault-Tec-like logos or vault jumpsuits
- Brotherhood, NCR, Enclave, Super Mutants, or other named Fallout factions
- SPECIAL as a named system
- exact perk names or perk icons
- exact dialogue lines
- exact maps or town layouts
- exact creatures such as specific branded mutants
- exact armor silhouettes unique to another game
- exact weapon names unique to another game
- exact sound effects or music
- extracted screenshots or textures as assets

## Safer alternatives

Instead of a vault jumpsuit:

- use quarantine work coats
- road-worker coveralls
- chapel field dressings
- scavenged settlement uniforms
- old medical isolation gear

Instead of a Pip-Boy-like device:

- use a cracked field ledger
- wrist-mounted quarantine tally counter with different silhouette
- brass-and-glass road compass
- paper map and mechanical stamp book
- shoulder-mounted radio slate

Instead of Fallout factions:

- road wardens
- ash chapel clergy
- water ration clerks
- rust engineers
- burial guild
- candle census office
- signal monks
- scrap law court

Instead of SPECIAL:

- use original attribute names or a different structure
- e.g. Body, Sense, Nerve, Charm, Mind, Motion, Fortune
- or use profession traits and survival disciplines

## Style vs. content

Allowed:

```text
“Use a late-90s isometric CRPG presentation with low-res pre-rendered-style sprites, muted colors, and heavy UI.”
```

Not allowed:

```text
“Make this exact UI look like Fallout 2 with the same buttons, borders, icons, Pip-Boy, perks, and faction logos.”
```

## Originality checklist

Before approving any asset, ask:

1. Would a player identify this as a direct copy of a known game asset?
2. Does it use a protected faction symbol, mascot, UI device, or exact silhouette?
3. Does it copy a map, quest, dialogue line, or character concept too closely?
4. Could the same style goal be achieved with a more original cultural or material design?
5. Is the design rooted in this game's own worldbuilding?

If yes to 1–3, redesign.

## Best practice

Use multiple references beyond one game:

- late-90s CRPGs generally
- old industrial equipment
- abandoned religious sites
- quarantine signage
- military surplus
- early computer terminals
- desert ruins
- analog medical equipment
- Soviet/Cold War industrial design
- rusted service corridors
- handmade settlement objects

The more real-world and cross-genre references used, the more original the result becomes.
