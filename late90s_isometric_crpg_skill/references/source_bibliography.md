# Source Bibliography and Reference List

This list is for research and calibration. It is not permission to copy assets.

## Agent skill format references

- Anthropic Claude API Docs — Agent Skills overview. Describes skills as `SKILL.md`-based packages with optional bundled reference files that can be loaded as needed.
  - https://platform.claude.com/docs/en/agents-and-tools/agent-skills/overview

- Anthropic Engineering — “Equipping agents for the real world with Agent Skills.” Explains the anatomy of a skill directory, `SKILL.md`, YAML metadata, and bundled resources.
  - https://www.anthropic.com/engineering/equipping-agents-for-the-real-world-with-agent-skills

- Anthropic public skills repository. Demonstrates self-contained skill folders with `SKILL.md` files.
  - https://github.com/anthropics/skills

- Claude Code Docs — Skills. Explains personal/project skills, `SKILL.md` frontmatter, and optional templates/examples/scripts/reference docs.
  - https://code.claude.com/docs/en/skills

## Technical rendering / pixel art references

- Unity 2D Pixel Perfect package docs. Covers point filtering, no compression, consistent Pixels Per Unit, and pixel-perfect camera setup.
  - https://docs.unity3d.com/Packages/com.unity.2d.pixel-perfect@latest/

- Unity blog: 2D Pixel Perfect setup. Notes that other filter modes can blur pixel art and compression can alter small sprites/palettes.
  - https://unity.com/blog/games/2d-pixel-perfect-how-to-set-up-your-unity-project-for-retro-8-bits-games

- Aseprite sprite sheet docs. Covers sprite-sheet export patterns: horizontal, vertical, matrix, tags, and atlas concepts.
  - https://www.aseprite.org/docs/sprite-sheet/

- Aseprite exporting docs. Covers exporting sprites/animations as GIFs or PNG sequences.
  - https://www.aseprite.org/docs/exporting/

- Blender manual: perspective/orthographic projections. Explains setting a camera to orthographic.
  - https://docs.blender.org/manual/en/latest/editors/3dview/navigate/projections.html

- GDQuest: Configuring Godot 4 for pixel art. Covers nearest-neighbor filtering and why linear filtering blurs pixels.
  - https://www.gdquest.com/library/pixel_art_setup_godot4/
